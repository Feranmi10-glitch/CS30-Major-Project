# project description
My project will be about a word game where you have five letters to choose from to make 3 3-letter words and a five letter word within 5 minutes.

## Needs To Have List 
- sounds/music
- boxes to insert letters
- a creative background
- at least 10 levels
- class
- timer

### Nice To Have List
- score system
- infinite levels
- bonus words
- 2 grid array
